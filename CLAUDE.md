# Project Conventions

## Stack

- React (functional components only, no class components)
- Tailwind CSS for all styling (no custom CSS files unless necessary)
- Vite as build tool

## Conventions

- Commit format: Conventional Commits (e.g., `feat: add navbar component`, `fix: resolve button alignment`)
- Code style: 2-space indentation, single quotes for strings
- Components go in `src/components/`
- Keep components small and single-responsibility
